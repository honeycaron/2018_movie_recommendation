# 2018_movie_recommendation
2018 자연어처리 기반 영화 추천 서비스

# 실행 방법 
pip에서 Flask 모듈을 설치 후, (그밖에 필요모듈: koNLPy, scipy)

cmd창에서 python demo.py로 활성화시킨 다음,

localhost로 접속
